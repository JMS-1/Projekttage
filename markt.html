Projekttage 2015 im Gymnasium am Ölberg in Oberpleis

Interaktive Web Seiten mit JavaScript - Übersicht für den Projektmarkt

In diesem Projekt wollen wir lernen, wie man in HTML mit CSS und vor allem JavaScript Web Anwendungen bereits im Web Browser (zum Beispiel Internet Explorer, FireFox oder Chrome) gestalten kann. Auch ohne die Hilfe von Programmen auf einem Web Server (zum Beispiel PHP Scripte) kann der Anwender nicht nur bei dem Ausfüllen von Formularen unterstützt werden. Vielmehr erlauben es auch schon einfache optische Effekt, die Aufmerksamkeit der Benutzer gezielt auf Teile der HTML Seite zu lenken oder eine HTML Seite einfach etwas ansprechender zu gestalten.
Zur Einführung in das Projekt werden wir uns erst einmal kurz anschauen, was man schon alleine mit HTML und CSS erreichen kann. Danach geht es allerdings schon sehr schnell in die ersten kleinen Beispiele der JavaScript Programmierung. Dabei geht es dann vor allem um die Möglichkeit, aus einem Programm heraus die HTML Seite zu verändert - vielleicht nur durch Veränderung der CSS Eigenschaften einzelner HTML Elemente aber auch durch Veränderungen der HTML Seite an sich.
Die meiste Zeit des Projektes werden wir uns mit der JavaScript Programmierung beschäftigen. Als Beispiel dient dazu erst einmal Das Spiel des Lebens. Dabei handelt es sich nicht wirklich um ein Spiel, sondern eher eine Simulation. Je nachdem wie schnell wir im Projekt vorankommen, können wir und auch noch an dem Spiel Schiffe versenken versuchen. Dabei wird das bis dahin Angewendete weiter vertieft - bis hin zu einigen praktischen Techniken zur Entwicklung von größeren JavaScript Web Anwendungen.
